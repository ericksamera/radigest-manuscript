# radigest manuscript reproducibility workflow

This repository contains the reproducible scaffold for the `radigest` manuscript workflow.

The current scaffold regenerates project configuration, environment specification, `radigest` build/test provenance, supported-enzyme snapshots, immutable-input checksums, vendored-source checksums, and workflow provenance.

It intentionally does not yet implement digest validation, enzyme screening, TLEN extraction, empirical model fitting, transfer analysis, benchmarking, tables, or figures.

## Runtime

Create or activate the manuscript environment:

```bash
mamba activate radigest-manuscript
````

The environment is specified in:

```text
envs/environment.yml
conda-lock.yml
```

## Scaffold workflow

Run the scaffold/provenance workflow with:

```bash
snakemake --cores 1 --printshellcmds
```

The main scaffold outputs are written under:

```text
results/provenance/
```

## Raw inputs

Reference FASTA and future raw FASTQ files are not tracked by Git. They are treated as immutable inputs and tracked by SHA256 in:

```text
results/provenance/input_checksums.sha256
```

Current reference paths:

```text
resources/references/trichoderma/ref.fa.gz
resources/references/salmon/ref.fa.gz
```

## radigest source

The `radigest` source is vendored under:

```text
resources/software/radigest/
```

The vendored source snapshot is documented in:

```text
resources/software/radigest/SOURCE_REVISION.txt
results/provenance/radigest.source_revision.txt
results/provenance/radigest.source_tree.sha256
```

Build products under `resources/software/radigest/bin/` are regenerated by Snakemake and are not tracked by Git.
